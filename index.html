News and Updates

06-01-2025

So far, I have created a basic HTML structure, and added some basic in-line CSS. I did create a style sheet that I will move the CSS to once it comes up in my course. Although it is tempting, I do not want to jump too far ahead.

06-06-25

Today I added links to other websites, placeholders. I added a poorly formatted placeholder image to be a logo for the website. Although it hasn't come up in my class yet, I added an unordered list as well. I added a link to the project license in the footer.

Today I added an auto-playing local video file to the header. I am also going to embed a Youtube video of the HTML tutorial that I am watching in conjunction with making this website, which should be embedded below.

I have modified the list in the nav area to be a description list, and gave each nav link a brief description. A new table is implemented that acts as a very basic roadmap of my progress thus far.

06-015-25

Today I learned about divs and spans, which I could not find a good place to implement, especially since I have decided that I will not be implementing a stylesheet. This project is a great way to practise HTML tags, but styling this jumbled mess would be nearly impossible. I added a non-functioning input form for website feedback, just a mock up to get familiar with the buttons really. I *tried* to add an iFrame of the development and learning roadmap website I am following, but the website does not allow iFrames, as most do not for security reasons. I left it implemented for demonstration purposes. Finally I added some more meta data, including viewport settings for mobile users.

This will likely be my last update to this website, as next week I will begin my course on CSS, and for that purpose I would like to implement a new html structure with content that would actually look decent when styled. I learned and re-learned a lot here though, and will probably keep this website hosted on my new one for reference.
